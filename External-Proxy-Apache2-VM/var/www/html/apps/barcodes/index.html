Sample Healthcare GS1 DataMatrix Symbologies

This page contains a series of GS1 DataMatrices that carry a GTIN (only) of a pharmaceutical product. Each one has at least one record in the GS1 resolver and so can be scanned using the SoomPharma app for iOS, or barValid* for Android to access the relevant patient information. Some entries have multiple links in the resolver. You can see the full list of what's in the resolver by clicking the link beneath each GS1 DataMatrix.

*barValid is not specifically a pharma-related app. At the time of writing it offers support for GS1 Digital Link as an experimental feature that has to be enabled in the app. It will deliver the default page for all entries in the GS1 resolver (it doesn't (yet) offer the option to select a link type). It can read all symbologies. Where no URL is encoded (as in the examples above) it will construct a GS1 Digital Link URI on id.gs1.org. It only recognises GS1 Digital Link URIs in QR codes that are on that domain. I does not, for example, recognise https://dalgiardino.com/gtin/09506000134352.

DataMatrices generated using Metafloor's Online Barcode API which is based on Terry Burton's Barcode Writer in Pure PostScript.
