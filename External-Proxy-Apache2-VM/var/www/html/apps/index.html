*UPDATED SEPT 2019* GS1 Resolver Barcode scanner

The GS1 Resolver Barcode Scanner App is a fully featured reference app for demonstrating the GS1 Digital Link standard, and is for Android devices. It scans product GTINs and looks up information, if available, on GS1's global ID resolver at id.gs1.org using the compatible GS1 Digital Link format:

https://id.gs1.org/gtin/{13-or-14-digit-gtin}/{optional digital link attributes in URI format}?{querystrings}

Using an onscreen 'Information Type' selector, you can choose from a range of 'Link Types' of information for the product, such as 'Product Description', 'Related video', and 'Recipe website' which are found in the GS1 Resolver's '.well-known' link at https://id.gs1.org/.well-known/gs1resolver.json If that information type is found when you scan the product, the app immediately redirects you to an appropriate third-pary page owned by that GTIN's member licensee, or else shows you a product information and links page from id.gs1.org itself.

For Data matrix barcodes, this app searches for and extracts these elements where they are present:
  • GTIN (01)
  • Expiry date (17)
  • Batch/lot number (10)
  • Serial number (21)

This information is used to form a URI of the following form:

https://id.gs1.org/gtin/{gtin}/lot/{lot}/ser/{ser}?expiryDate={expiry date}

You can use the app to try out 1D Barcodes, Data Matrix barcodes, and QR Codes containing links in GS1 Digital Link format. Try these samples: https://philarcher.org/gs1/2019/qrSamples/

Created for demonstrations for and by GS1 Member Organisations. Requires the android operating system on your mobile phone only at this time. For demonstration purposes only

*UPDATED SEPT 2019* GS1 Resolver 2019 edition scanner

The GS1 Resolver 2019 edition scanner is an updated app for Android devices that scans a GS1 DataMatrix (as well as 1D barcodes). The updated app asks the resolver for available linktypes and has been updated to include the Norewegian language option. The app follows several user stories for both healthcare and supermarket products. For Data matrix barcodes, this app searches for and extracts these elements where they are present:

  • GTIN (01)
  • Expiry date (17)
  • Batch/lot number (10)
  • Serial number (21)

This information is used to form a URI of the following form:

https://id.gs1.org/gtin/{gtin}/lot/{lot}/ser/{ser}?expiryDate={expiry date}

You can test the app using Data Matrix images at https://philarcher.org/gs1/2019/brusselsdemos/

Created for January and February 2019 events in Brussels, and updated for demonstration ebvents in Autumn 2019 - and is to be used for demonstration purposes only.

GS1 DataMatrix scanner

The Healthcare demo is an app for Android devices that scans a GS1 DataMatrix. It extracts the elements usually found in the Data Matrix code string:

  • (01) GTIN
  • Expiry date (17)
  • Batch/lot number (10)
  • Serial number (21)

    • This information is used to form a URI of the following form:

    https://id.gs1.org/gtin/{gtin}/lot/{lot}/ser/{ser}?expiryDate={expiry date}

    For the demo, the server will either redirect to a pharmaceutical manufacturer's product description page (the demo was created in association with GSK) or will perform an FDA GUDID lookup live using their API. You can test the app on the Data Matrix images at https://id.gs1.org/apps/hctestimages
